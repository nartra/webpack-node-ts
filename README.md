Webpack 4 + TypeScript starter
------------------------------

- [x] Webpack 4
- [x] TypeScript 2 
- [x] Webpack Development Server

## How to use

### install all dependencies.
```
$ npm i
```

### Start developing and serve your app:
```
npm start
```

### Build your app without minification:
``` 
npm run build
```

### Build your app with minification:
``` 
npm run build:prod
```
